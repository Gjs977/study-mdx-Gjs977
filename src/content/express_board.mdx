# Express Board 📝

Node.js와 Express를 활용해 만든 **팀 프로젝트 게시판 사이트**

- [GitHub 저장소 보기](https://github.com/deadrabbit07/Board)
- [사이트 바로가기](http://3.91.222.159:3000/board.html)

![생성된 계획](/img/BSC.png)

## 사용 방법

1. 🔐 회원가입 및 로그인
2. 📝 게시글 작성
3. 📋 전체 게시글 목록 확인
4. 🔍 게시글 상세보기
5. ✏️ 게시글 수정 / 삭제

---

## 기술 스택

- Backend: **Node.js**, **Express**
- Frontend: **HTML**, **CSS**, **JavaScript**
- 협업 도구: **Figma**, **Notion**, **GitHub**
- 배포: **Docker**, **AWS EC2**

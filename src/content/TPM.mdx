# TourPlanMate ✈️

AI가 대신 여행 계획을 짜주는 웹사이트

![메인 페이지](/img/TPM_main.png)

---

## 사용 방법

1. ✍️ 원하는 여행지 입력
2. 🎯 하고 싶은 활동 입력
3. ⏳ 여행 기간 선택
4. 📅 출발 날짜 선택
5. 💰 예산 범위 선택
6. 👥 여행자 수 선택
7. 🧠 **[여행 계획 생성하기]** 버튼 클릭

---

## 기술 스택

- Frontend: **HTML**, **CSS**, **JavaScript**
- Backend: **Flask**
- AI: **ChatGPT API**

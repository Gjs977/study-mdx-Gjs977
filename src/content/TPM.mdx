# ✈️ TourPlanMate

AI가 대신 여행 일정을 짜주는 스마트 웹사이트

[🔗 GitHub 저장소](https://github.com/Gjs977/TourPlanMate)  
[🌐 서비스 바로가기](http://54.205.44.155:3000)

![메인 페이지](/img/TPM_main.png)

---

## 📌 주요 기능

1. 여행지 및 활동 입력
2. 여행 기간 및 날짜 선택
3. 예산 및 인원수 설정
4. 👉 **[여행 계획 생성하기]** 버튼 클릭 시, AI가 일정을 자동으로 생성

---

## 🛠 기술 스택

- Backend: **Python**, **Flask**
- Frontend: **HTML**, **CSS**, **JavaScript**
- AI: **OpenAI**, **ChatGPT API**
- 배포: **AWS EC2**

## 💡 프로젝트 개요

- **목표**: 사용자 맞춤형 AI 여행 계획을 자동으로 제공
- **성과**:
  - ChatGPT API와 Flask를 연동해 실제 사용 가능한 여행 생성 서비스 구현
  - 사용자 입력 기반의 유연한 일정 생성 알고리즘 설계
- **기여도**: 100% (기획, 디자인, 개발, 배포)

---

## 🧠 사용 기술 요약

- `Flask`: 백엔드 서버, API 통신 담당
- `ChatGPT API`: 사용자 입력 기반으로 여행 계획 생성
- `Vanilla JS`: UI 로직 구현
- `AWS EC2`: 서버 배포 및 운영
